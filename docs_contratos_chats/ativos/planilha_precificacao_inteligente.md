Este documento serve como guia para o cálculo de orçamentos de projetos de consultoria e desenvolvimento.

## 1. DEFINIÇÃO DO VALOR HORA
Calcule seu valor hora com base nos seus custos e objetivos:
- **Renda Mensal Desejada:** R$ 10.000,00
- **Fator de Custos (Impostos, Software, Infra):** 2x
- **Horas Faturáveis Mensais:** 120h
- **Valor Hora Calculado:** (10.000 * 2) / 120 = **R$ 166,67 / hora**
